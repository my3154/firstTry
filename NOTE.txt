VIM

ll 查看当前目录下所有非隐藏文件

ls -la查看此目录下所有文件(包括隐藏的)

git init 创建本地git库

设置签名
    用户名 || Email地址 注意，此处设置的签名和GITHUB无关!!!只是为了区别开发者身份
    项目/仓库级别：在有.git文件内的范围生效
    git config user.name My3154
    系统用户级别：登录当前操作系统的范围 就近原则优先用项目级别
    git config --global user.name My3154
    信息会保存在.git下的config文件内

git status 
    查询此文件夹下的文件状态
